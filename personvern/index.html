Personvernerklæring

Denne siden forklarer hvordan Musta Digital behandler personopplysninger når du kontakter oss eller bruker nettsiden.

Behandlingsansvarlig

Musta Digital er behandlingsansvarlig for personopplysninger som samles inn via nettsiden og kontaktskjemaet.

Kontakt: kontakt@mustadigital.no

Hvilke opplysninger vi samler inn

Når du sender inn kontaktskjemaet, kan vi lagre navn, e-postadresse, firmanavn, valgt tjeneste og meldingen du sender inn. Hvis du kontakter oss på e-post, kan vi lagre innholdet i dialogen for å kunne følge opp henvendelsen.

Hvorfor vi bruker opplysningene

Vi bruker opplysningene for å svare på henvendelser, følge opp potensielle kunder, levere tjenester, føre enkel kundehistorikk og forbedre kommunikasjonen med deg.

Behandlingsgrunnlag

Behandlingen skjer normalt fordi det er nødvendig for å svare på en henvendelse, for å gjennomføre tiltak før en eventuell avtale, eller fordi vi har en berettiget interesse i å følge opp kundehenvendelser.

Lagring og sletting

Vi lagrer opplysninger så lenge det er nødvendig for formålet de ble samlet inn for. Du kan be oss slette eller rette opplysninger vi har om deg, med mindre vi er pålagt å beholde dem av regnskapsmessige eller andre lovpålagte grunner.

Hvem opplysningene deles med

Opplysninger kan behandles av leverandører som hjelper oss med nettside, e-post, hosting, database og analyse. Eksempler kan være Vercel, e-postleverandør og Supabase dersom database er aktivert. Vi selger ikke personopplysninger til andre.

Dine rettigheter

Du kan be om innsyn, retting, sletting, begrensning eller protestere mot behandling av opplysninger. Du kan også klage til Datatilsynet hvis du mener behandlingen bryter personvernregelverket.

Analyse og tekniske data

Nettsiden kan bruke enkel analyse for å forstå besøk og ytelse. Slike data brukes til å forbedre nettsiden og driften.

Endringer

Denne personvernerklæringen kan oppdateres når tjenestene eller systemene våre endres.
